Plugins â€‹

  • vite-plugin-electron makes the integration of Electron With Vite very easy! Its main purpose is to build the Main process and Preload scripts, and to start and hot restart the Electron App when the Main process and Preload scripts are built.

    It also provides a full JavaScript API to make it easy to use anywhere.

  • vite-plugin-electron-renderer makes it possible to support building Electron Renderer process by modifying some of the necessary Vite default configs.

    In addition it allows some npm packages developed specifically for Node.js, especially C/C++ native modules, to be built properly by Vite to be available to the Renderer process.
